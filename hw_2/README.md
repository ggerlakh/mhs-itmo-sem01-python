## Создание виртуального окружения

1. Окружение создается через `conda`

   ```bash
   conda create -n hw_2 -c conda-forge python=3.12 pdflatex 
   ```
2. Для активации окружения используется следующая команда:
   
   ```bash
   conda activate hw_2
   ```
3. Для деактивации:

   ```bash
   conda deactivate
   ```
