# Модуль по работе с матрицами

В файле `matrices.py` описан небольшой модуль по работе с матрицами (`class Matrix`) в котором реализованы три операции:
- поэлементое сложение (`+`)
- поэлементое умножение (`*`)
- матричное умножение (`@`)
